#  Movie App

App to search trending and specific movies

Live demo: https://movie-app-puce-five.vercel.app/

## How to run the App

```pnpm install```

Then

```npm run dev```

## Tech stack

 - TailwindCSS
 - Vite
 - React
 - React Router DOM
 - Api Themoviedb

 ## Improvements
